Install react-native-maps

Install react-native-maps official site
you could choose the above one or mine to install react-native-maps

  1. Run “android” and make sure every packages is updated.

  2. I am using genymotion so I need to do some install before install google play:

    • Install ARM Translation,download Genymotion-ARM-Translation_v1.1zip,then drag and drop the downloaded file to an opened Genymotion virtual device. After file transfering is done, there will be a confirmation dialog ,just click OK all the way. You need to reboot the virtual device now but we don’t suggest to just close the virtual device and relaunch it once again since it may leads to some weird behaviour.. First we need to reboot the device. cd C:\Users[yourUserName]\AppData\Local\Android\sdk\platform-tools Now close the device and restart it again
      In case you accidentally close the virtual device, don’t be panic. You might need to relaunch for a few times before it can boot up.
    • Install gapps. download the flashable zip of gapps by your virtual device’s Android version:Android 5.0.x, Android 4.4.x, Android 4.3.x, Android 4.2.x, Android 4.1.x, Android 4.0.x, Android 2.3.3 and thendrag and drop the downloaded file to the virtual device and like before reboot the device.
    • “Google Play services has stopped” will keep popping. don’t worry. just go to google play and update all apps.

  3. Install the following packages in android SDK manager :

    • Extras / Google Play services
    • Extras / Google Repository
    • Android 6.0 (API 23) / Google APIs Intel x86 Atom System Image Rev. 13

  4. Generate your SHA1 key

    • cd C:\Program Files\Java\jdk1.8.0_77\bin where your keytool.exe exsit
    • type ,
    • to show the key: type . it would be like this “D8:AA:43:97:59:EE:C5:95:26:6A:07:EE:1C:37:8E:F4:F0:C8:05:C8”
    • find your packname from file C:\Users[yourusername]\AwesomeProject\android\app\src\main\AndroidManifest.xml
    • Go to Google API Console and select your project, or create one.Create a new key by clicking on Create credentials -> API Key -> Android Key,enter the name of the API key and your SHA1 key and create it. Then we get android API key

  5. Manual install

    • in your file build.gradle add: dependencies { ... compile 'com.airbnb.android:react-native-maps:0.7.1' }
    • Add your API key to your manifest file:
  • For React Native v0.29.0 or above:in your application object which located C:\Users[username]\getAwesome\android\app\src\main\java\com[projectName], add: public class MyApplication extends Application implements ReactApplication { private final ReactNativeHost reactNativeHost = new ReactNativeHost(this) { @Override protected List getPackages() { return Arrays.asList( new MainReactPackage(), new MapsPackage()); } }; @Override public ReactNativeHost getReactNativeHost() { return reactNativeHost; } }

problems:

FAILURE: Build failed with an exception.
What went wrong:
A problem occurred configuring project ‘:app’.
Cannot evaluate module react-native-maps : Configuration with name ‘default’ not found.

Solution: edit file C:\Users[yourUsername]]\AwesomeProject\android\settings.gradle change to

include ‘:react-native-maps:lib’
project(‘:react-native-maps’).projectDir = new File(rootProject.projectDir, ‘../node_modules/react-native-maps/android/lib’)

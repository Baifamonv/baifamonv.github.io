what make React Native special?

“The idea of React Native meant we could bring all the things we loved about developing for the web — such as fast iterations and having a single team build the whole product — to mobile. It meant we could move faster”
At the same time
“..to ship the first fully React Native-powered app where the user experience was identical to an app built in Objective-C”
“..having to build the same app separately for each platform is a problem in the industry”

Martin Konicekcode.facebook.com/posts/597378980427792/react-native-a-year-in-review

React Native has rapid adoption and vibe community and release almost every two weeks. ‘..code was downloaded about 70,000 times from npm in March 2016’. Popular applications like airbNb and qq are created uing React Native.

Beyond apps, there is a wide variety of services that build upon or complement React Native.Exponent lets you build and share React Native apps without having to compile anything; React Native Playground lets you edit and run React Native apps in the browser; AppHub and Microsoft CodePush let you deploy code instantly without going through the app stores; JS.coach is a database of third-party modules; and Deco is an IDE for React Native.

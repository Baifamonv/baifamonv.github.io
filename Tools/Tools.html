Exponent

Exponent is a mobile app development tool that enables you to build high quality native iOS and Android applications using JavaScript and deploy them effortlessly.
Writing an Exponent app is like writing a React Native app, except you never need to open Xcode / Android Studio, or write any native code. The Exponent SDK provides access to system functionality such as contacts, camera and social login
